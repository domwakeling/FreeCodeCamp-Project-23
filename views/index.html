URL Shortener Microservice

Creating a new shortened URL

To create a new shortened URL, enter the address https://doms-url.herokuapp.com/new/ followed by the URL that you wish to shorten.

So, for example, to generate a shortcut for http://www.domwakeling.com you would use this address in the browser address bar:

https://doms-url.herokuapp.com/new/http://www.domwakeling.com

The URL that you are shortening must start with either http:// or https://.

Result

You will receive a JSON with the shortened URL, so for instance:

{'long_URL': 'http://www.domwakeling.com/', 'short_URL': 'https://doms-url.herokuapp.com/1'}

Using a shortened URL

To use a shortened URL, enter it into the address bar.
