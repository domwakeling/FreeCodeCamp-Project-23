URL Shortener Microservice

Creating a new shortened URL

To create a new shortened URL, enter the address https://dry-castle-31853.herokuapp.com/new/ followed by the URL that you wish to shorten.

So, for example, to generate a shortcut for http://domwakeling.com you would use the full address:

https://dry-castle-31853.herokuapp.com/new/http://domwakeling.com

In return you will receive a shortened URL, for instance

https://dry-castle-31853.herokuapp.com/1

If a shortened URL already exists for the website, that URL will be returned; otherwise a new shortened URL will be constructed and returned.

Using a shortened URL

To use a shortened URL, enter it into the address bar.
